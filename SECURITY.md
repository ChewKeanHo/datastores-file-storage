# Security Policy - Reporting a Vulnerability

[![chewkeanho](/.internals/icons/animated-banner_1200x100.svg)](#)

We take security issues seriously AND WITHOUT CAUSING PUBLIC PANICS. Therefore,
if you find any security vulnerability (both suspicious or confirmed), please
refer to the following link for the procedures:

[(Holloway) Chew Kean Ho's Security Section](https://github.com/chewkeanho/security)

**DO NOT TO CAUSE PUBLIC PANICS BY PUBLIC DECLARATIONS. Please be professional.**
